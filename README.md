
# E-Commerce Sales Performance Dashboard (Excel)

##  Project Overview
This project is an interactive E-Commerce Sales Performance Dashboard developed in Microsoft Excel.  
The dashboard provides insights into sales and profit trends across product categories, regions, and months using dynamic visualizations.

---

##  Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers  
- Combo Box (Form Controls)  
- SUMIFS, INDEX formulas  
- Data Cleaning & Transformation  
- Interactive Dashboard Design  

---

##  Key Features
- Category-based dynamic analysis using Combo Box  
- Regional sales distribution with Pivot Charts and Slicers  
- Monthly sales trends using column and line charts  
- Sales vs Profit analysis using scatter chart  
- Clean, professional, and interview-ready dashboard layout  

---

##  Dashboard Insights
- Analyze monthly sales performance  
- Compare regional sales across different product categories  
- Identify sales and profit relationships  
- Interactive filtering for better decision-making  

---

##  Project Structure
- `Data` – Raw sales data  
- `Working Sheet` – Calculations and helper tables  
- `Dashboard` – Final interactive dashboard  

---

##  How to Use
1. Download the Excel file  
2. Open the **Dashboard** sheet  
3. Use the **Category slicer or combo box** to interact with the dashboard  
4. Charts will update automatically based on selection  

---

##  Use Case
This project demonstrates practical Excel skills required for data analysis roles, including dashboard creation, data visualization, and business insights generation.
